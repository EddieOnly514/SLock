# SLock

**A social app to help reduce screen time on addictive apps**

Lock distracting apps, earn points, and redeem rewards like gift cards. Stay accountable with friends through social circles.

---

## Features

- **App Locking** - Block distracting apps during focus sessions
- **Rewards** - Earn 100 points/hour for locked time, redeem for gift cards (Amazon, Starbucks, etc.)
- **Social Circles** - Join friends in accountability groups with shared focus rules
- **Leaderboards** - Compete with friends on screen time and streaks

---

## Tech Stack

- **Framework**: React Native with Expo
- **Language**: TypeScript
- **Navigation**: Expo Router
- **Backend**: Firebase

---

## Getting Started

```bash
cd SLock
npm install
npm start
```
